# 看电视 (WatchTV)

一个基于 Jetpack Compose 开发的 Android 影视点播应用。

## 🌟 项目简介

本应用接入了量子资源 (Liangzi Ziyuan) 的标准接口，实现了流畅的在线观影体验。UI 设计遵循现代 Material 3 规范，并针对影视类应用进行了“大片感”视觉优化。

## 🚀 核心功能

- **影视点播 (VOD)**：实时接入 `lzm3u8` API，支持海量影视资源在线播放。
- **智能搜索**：支持按影片名、导演、演员进行全局搜索。
- **分类筛选**：优化后的频道分类，自动过滤无资源类别，支持热门频道优先展示。
- **沉浸式观影**：
  - 集成 **Media3 (ExoPlayer)**，支持 `.m3u8` 格式流媒体播放。
  - **全集选集**：自动解析多集数据，支持点击切集。
  - **自动续播**：观看结束后自动跳转至下一集。
  - **多线路切换**：智能过滤并展示有效播放来源。
- **双模式全屏**：
  - 支持 **横屏全屏** (适合电影/剧集)。
  - 支持 **竖屏全屏** (适合短剧/短视频)。
  - 支持系统自动旋转与手动按钮强制切换。
- **极致视觉**：
  - 首页“剧场版”海报墙设计，带渐变阴影文字和状态角标。
  - 搜索框采用圆角胶囊设计，UI 交互自然丝滑。
- **性能优化**：
  - **分页加载**：支持无限滚动，滚动到底部自动补全后续数据。
  - **响应式状态同步**：切换分类自动清空搜索，确保数据展示最大化。

## 🛠️ 技术栈

- **语言**: Kotlin
- **UI 框架**: Jetpack Compose (Material 3)
- **网络**: Retrofit + OkHttp + Gson
- **图片加载**: Coil
- **播放器**: Media3 (ExoPlayer) + HLS Support
- **架构**: MVVM (ViewModel + Flow)

## 📥 安装与运行
